# Asset Sources

- Pack: Tower Defense
- URL: https://kenney.nl/assets/tower-defense

- Pack: Isometric Roads
- URL: https://kenney.nl/assets/isometric-roads
- Usage: isometric ground and road tiles, tree decoration

- Pack: UI Pack
- URL: https://kenney.nl/assets/ui-pack
- Usage: full HUD skin (panels, buttons, bars) and UI fonts

- Pack: Particle Pack
- URL: https://kenney.nl/assets/particle-pack
- Usage: projectile and hit VFX sprites
